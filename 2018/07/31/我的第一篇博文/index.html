我的第一篇博文-基于GitHub Pages的博客搭建

今年的5月份开始学习前端，那时候就想着以后要搭建一个自己的博客。今天终于开始动手搭建了！

基础信息：

  • 系统：windows 10 64位
  • 编辑器：sublime text3
  • 控制台：Cmder
  • 博客框架：Hexo

    Hexo博客框架

    快速、简洁且高效的博客框架。进入Hexo官网

配置Github

Create a new repository
Repository name 为： yourname.github.io (yourname需替换成你的注册用户名,这个就是你博客的域名了)
例如，我的Github主页是github.com/xingteng。就填入xingteng.github.io。

环境安装

  1. 安装Node.js

  2. 安装Git for Windows–国内站点.对Git设置用户名和Email（请自行baidu或Google）

  3. 安装Cmder-解压后即可使用

  4. 打开Cmder输入以下命令
